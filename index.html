Card Generator

A high-performance REST API to generate stunning glassmorphism character cards for Xian Ni (Renegade Immortal) WhatsApp Bots & Web Apps.

Endpoint & Usage

Generate a dynamic image by calling the API endpoint. You can use predefined character presets from the database or build custom characters via URL queries.

/api/card?character=wang-lin&apikey=YOUR_API_KEY

Parameters

Parameter Description
apikey Kunci akses keamanan wajib
character ID karakter preset (ex: wang-lin, situ-nan). Jika dipakai, otomatis mengisi stats.
name Nama karakter custom (ex: Teng Huayuan)
rarity Divine | Legendary | Epic | Rare | Uncommon | Common
atk, def, hp Statistik angka karakter (ex: 9999)
emoji Emoji utama di tengah kartu jika gambar gagal (ex: ❤️)
image URL gambar kustom (opsional)
json Set true untuk mengambil data mentah JSON tanpa gambar (Sangat berguna untuk menyusun caption teks pada Bot WhatsApp).

Examples

  • Preset Image: /api/card?character=li-muwan&apikey=YOUR_API_KEY
  • Raw JSON Data: /api/card?character=wang-lin&json=true&apikey=YOUR_API_KEY
  • Override Stat: /api/card?character=situ-nan&atk=999999&apikey=YOUR_API_KEY
  • Full Custom: /api/card?name=Dewa&rarity=Divine&atk=10&def=10&hp=10&apikey=YOUR_API_KEY
